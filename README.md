# タスク管理アプリの作成

## 概要
このプロジェクトはReactとSpringBootの学習を並行して行うことを目的として作成しました。

## 使用技術
- React
- SpringBoot
- PostgreSQL
- BCrypt
- JWT

## 機能
- ユーザー登録機能
- ログイン・ログアウト機能
- タスク登録・削除・編集機能
- カテゴリー登録・削除機能

## デモ
https://task-management-pink-eta.vercel.app/

## 工夫した点
- 認証にJWTを採用し、セッション管理ではなくトークンベースでログイン状態を維持する設計にしました。
- フロントエンドとバックエンドを分離し、API経由でデータ通信する構成にしました。
- パスワードはBCryptでハッシュ化し、平文保存を避けるようにしました。
